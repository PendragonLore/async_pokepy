Welcome to async_pokepy's documentation!
========================================

``async_pokepy`` is an asynchronous library for the `Pokeapi.co API <https://pokeapi.co>`_.

**Features:**
- Modern ``async``\/``await`` python syntax
- HTTP status codes handling
- Caching resources.
